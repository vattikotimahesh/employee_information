# Employee Information System

## Overview

Employee Information System is a console-based Python application developed to manage employee records efficiently. The project uses Python's file handling and `pickle` module to store employee data permanently in a binary file.

## Features

* Add new employee records
* View a specific employee's details
* View all employee records
* Search employee by Employee ID
* Update employee salary and company information
* Delete employee records
* Menu-driven interface for easy navigation
* Persistent data storage using Python Pickle

## Technologies Used

* Python
* File Handling
* Pickle Module
* Modular Programming

## Project Structure

* `Empmenu.py` – Displays the main menu.
* `Empmainproject.py` – Main driver program that controls the application flow.
* `EmployeeAdd.py` – Adds new employee records.
* `Employeedelete.py` – Deletes employee records.
* `Employeeupdate.py` – Updates employee information.
* `Employeeview.py` – Views single or all employee records.
* `Employeesearch.py` – Searches employee records.

## Employee Record Format

Each employee record contains:

* Employee Number (ID)
* Employee Name
* Employee Salary
* Employee Company Name
